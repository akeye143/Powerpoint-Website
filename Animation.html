ADD-ONS

Traditionally it has been incredibly difficult and complex to make a PowerPoint interactive. Classpoint is a PowerPoint add-on provides a diverse range of interactive quizzes such as mutiple choice, short answer and word cloud questions.

There is an automatic grading feature that can be used through the mutiple choises questions, this feature automates the grading process, includes an option to select difficulty levels, provide star ratings and export the quiz report.

This is the perfect tool for teacher to use for mid-term to get a grasp on the studnets knowledge and what topics need to be refreshed.
